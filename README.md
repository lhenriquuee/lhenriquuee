# Hi, I'm Luís Henrique ☕

**Technical Writer | Docs-as-Code | Markdown · GitHub · Git**

I turn complex, technical information into clear documentation that helps people actually get their job done — from product guides to API references.

## About me

I started my career in IT over 10 years ago, working in infrastructure and technical support for end users and corporate teams. That hands-on experience troubleshooting real problems is what pulled me into Technical Writing in 2021: I realized the documentation I wished I'd had was the documentation I wanted to build.

Today I focus on the **Docs-as-Code** methodology — writing in Markdown, versioning content in Git/GitHub, and managing documentation workflows through Jira alongside engineering and product teams.

**What I work on:**
- Product and feature documentation
- Tutorials, user guides, and help center content
- FAQs
- Release notes
- Internal developer and support documentation

In 2022, I presented an internal talk on *"The Value of Technical Documentation for Products,"* covering why documentation matters for product adoption and how a docs workflow fits into the development cycle.

I enjoy working cross-functionally — pulling insights from engineers, product managers, and support teams to turn scattered knowledge into documentation people can actually trust and use.
